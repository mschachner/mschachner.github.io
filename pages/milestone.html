Milestone

I made an abstract strategy game called Milestone, which became the subject of a computer science project by Connor McCarthy, Corban Chiu, and Sunny Chavan. They built a tree-search AI using an evaluation powered by hyper-heuristic tuning, and trained a neural network as well which played nearly half a million games! You can find their report here.

Below is a playable version, with rules and my own AI included. Please let me know about any bugs!
